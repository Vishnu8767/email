🚀 How to deploy on Railway
  1. Download all files → push to a new GitHub repository
  2. Go to railway.app → New Project → Deploy from GitHub
  3. Select your repo → Railway auto-detects Python + installs requirements
  4. In Railway dashboard → Variables → add: EMAIL_USER, EMAIL_PASS, NVAPI_KEY
  5. Railway gives you a live https://yourapp.railway.app URL instantly
